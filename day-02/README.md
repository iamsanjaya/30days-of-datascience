# Day 2 — Word Frequency & Primitive Sentiment Arc

## What I built

- Word frequency counter using `collections.Counter`, `map`, `filter`, `lambda`
- Primitive sentiment analyzer using manually curated word lists (no ML)
- Emotional arc plotted across 62 chapters of Pride and Prejudice

## Results

- Most positive chapter: 33 (Darcy's first proposal)
- Most negative chapter: 47 (Lydia's elopement)
- Highest tension chapter: 39

## Key learning

Smoothing is a display transformation, not a data transformation.
Peak detection must always use raw values.
